== Description ==

Este es un plugin que te ayudara a administrar publicidad pagada en tu sitio web. La forma de pago es Costo por Clic (CPC) y se hace a traves de Paypal o Cheque.

= Features =

- Creatividades listas para colocar en tu sitio web.
- Rentabiliza tu pagina en unos minutos.
- No hay restricciones, ni exigencias.
- No tienes que poner codigo.
- Es facil de instalar.

== Installation ==

Widget BUMBABlog Adserver is easy to install like other plug-ins of Wordpress as you need to just follow the below mentioned steps:

1. Copy Folder Music Bar from the downloaded and extracted file.

2. Paste it in wp-Content/plugins folder on your Wordpress Installation 

3. Activate the plugin from Dashboard / Plugins window.

4. Now Plugin is Activated.
